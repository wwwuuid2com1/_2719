操作指南<p>
<p>如何开始
游戏加载完毕后点击PLAY GAME – 再点击New Game – 然后选择角色 – 接着点击Fight即可开始游戏。
操作方法
鼠标点击两个相邻的武器图标交换位置，至少三个相同的武器图标连成一条直线即可消除哦。
提示：点击右侧的Suggest可获得提示(需冷却的时间)。
